    Html 5

    Sun 28 February 2016

    Hyper Text Markup Language (HTML) Version 5 is the current reality/future and you should use it when possible (avoiding horrible things like Adobe Flash).

    Simple Example

    <!DOCTYPE html>
<html>
<header></header>
<body>

        <p>Embed a PDF in an html page</p>
        <object width="400" height="400" data="test.pdf"></object>

</body>
</html>

    iFrame

    <!DOCTYPE html>
<html>
<body>

<iframe src="http://www.google.com" height="400" width="250"></iframe>

</body>
</html>
